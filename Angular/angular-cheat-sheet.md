##### Install Angular CLI

```
npm install -g @angular/cli
```

##### Update Anglular CLI

```
npm uninstall -g angular-cli @angular/cli

npm cache clean

npm install -g @angular/cli
```

##### Build Angular 5 application

```
ng build --env=local --watch
```

##### Rung Angular on Different port on run time

 ```
 ng serve --port ANOTHERPORT
 ```
